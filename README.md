# play-with-english
## CLI commands
```bash
# Project setup
yarn install

# Compiles and hot-reloads for development
yarn run serve

# Compiles and minifies for production
yarn run build

# Run tests
yarn run test

# Lints and fixes files
yarn run lint

# Run your unit tests
yarn run test:unit
```
